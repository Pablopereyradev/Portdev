Bienes Raices

El proyecto "Bienes Raíces" es una plataforma web diseñada para ofrecer a usuarios y potenciales clientes una experiencia intuitiva y completa en la búsqueda de propiedades tanto en venta como en alquiler.

enlace

Descripción del Proyecto

El proyecto "Bienes Raíces" representó un desafío que me permitió poner a prueba y desarrollar mis habilidades en el desarrollo de aplicaciones web utilizando el patrón de diseño Modelo-Vista-Controlador (MVC), lo que me resulto en una las mejores maneras de estructurar y organizar el código de la plataforma.

Con opciones de registro para vendedores y propiedades tanto en venta como en alquiler, los usuarios pueden agregar descripciones detalladas, imágenes y especificaciones como la cantidad de ambientes. Además, cuenta con un formulario de contacto funcional con validación, garantizando una comunicación efectiva entre interesados y vendedores.

La seguridad está garantizada mediante protección de URL según el usuario y acceso restringido para usuarios no logeados. Todo esto se ha desarrollado utilizando el patrón MVC, asegurando un código ordenado y escalable para el óptimo funcionamiento de la plataforma.

En resumen, el proyecto "Bienes Raíces" proporciona una solución completa y segura para la gestión de propiedades inmobiliarias, tanto para vendedores como para usuarios finales. La combinación de características robustas y una arquitectura MVC sólida garantiza un funcionamiento eficiente y una experiencia de usuario satisfactoria.
