Descripción del Proyecto

Este proyecto consistió en la creación de un blog para una cafetería, destacando varias publicaciones, una sección dedicada a cursos, y un formulario de contacto.

La implementación de GULP automatizó tareas, el diseño dio como resultando un sitio web completamente adaptable a diferentes dispositivos. Esta experiencia no solo fortaleció mis habilidades técnicas, sino que también marcó el comienzo de mi apasionante trayectoria en el campo del Desarrollo Web.
